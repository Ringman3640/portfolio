Event Testing Description

View the console dubugger in the browser to view the testing outputs.

There are events that trigger during specific time intervals that simply state how much time has passed. These events occur at:

1 second, 2 seconds, 3 seconds, 4.5 seconds, and 20 seconds.


To test simultaneous events, there is another event a 1 second that will execute at the same time as the previous event at 1 second. It has the following message:

"Simultaneous event at 1 second"

The "Load Media" button can also be pressed multiple times to reload the time events. Each time event message will be repeated simultaneously by the amount of time the button has been clicked.


To test continuous events, there is one continuous event that occurs at the very beginning of the timeline (0 seconds). There is another continuous event that occurs at 60 seconds. These events print a message to say if the current time of the timeline is before or after 1 minute. This is printed to the console whenever the user manually changes time.
